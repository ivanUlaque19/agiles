
## Agile Project

## Enviroment

- **Requirements**

  - python3.5 >
  - pip 
  - postgresql
  - virtualenv (with pip)

- **Folder project**

`$ mkdir agiles`

`$ cd agiles`

`$ virtualenv server`

- specify version python for virtualenv

`$ python3.6 -m venv server`

- **Clone project**

`$ git clone https://github.com/andru1236/agiles`

- **Run project**

`$ cd agiles`

- Active virtualenv

`$ source ../server/bin/activate`

`$ pip install -r requirements.txt`

`$ python manage.py runserver`
